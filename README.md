# dotbot

## How to use

First create a Discord Bot and get the Bot Token. Then create a file called .env. In this file put

```
TOKEN=YOUR_TOKEN_HERE
```

There is a Dockerfile. Use it.

```docker build --tag dotbot:1.0 .```
```docker run dotbot dotbot:1.0```
